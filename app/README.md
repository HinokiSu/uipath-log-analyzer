# Uipath Log Analyzer · Front-end
Uipath 日志解析与统计

## Quick Use
- Install
```bash
$ yarn
```
> Tips: 本项目使用 yarn 3.3
>  [yarn更新方法](https://yarnpkg.com/getting-started/install#updating-to-the-latest-versions)
> ``` bash
> $ yarn set version stable
> ````

- Run
```bash
$ yarn dev
```


 ## Features
- 日志状态统计
- 日志文件解析
- 日志文件状态
- 根据不同条件查看日志
